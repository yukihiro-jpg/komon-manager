Googleドライブ連携の設定手順
  1. Google Cloud Console でプロジェクトを作成
  2. 「Google Drive API」を有効化／OAuth同意画面を設定し自分をテストユーザーに追加
  3. OAuthクライアントID（ウェブアプリ）を作成し、承認済みJavaScript生成元に http://localhost:8080 を追加
  4. クライアントIDとフォルダURLを入力 →「設定を保存」→「ドライブに接続」
ローカルサーバで開く例： python -m http.server 8080 を実行し http://localhost:8080 を開く
※ file:// で直接開くとGoogle認証は許可されません。認証なしでもデータはブラウザに保存され、手動バックアップでドライブ保存／復元できます。
