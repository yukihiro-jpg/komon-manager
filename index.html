設定

Googleドライブ連携

このフォルダに komon-manager-data.json として登録内容を自動保存します。
Google Cloud Consoleで取得したOAuth 2.0 クライアントID。

手動バックアップ

Googleドライブ連携の設定手順を表示
  1. Google Cloud Console でプロジェクトを作成
  2. 「APIとサービス」→「ライブラリ」で Google Drive API を有効化
  3. 「OAuth同意画面」を設定し、自分のGoogleアカウントをテストユーザーに追加
  4. 「認証情報」→「OAuthクライアントID」を作成（種類: ウェブアプリケーション）
  5. 承認済みのJavaScript生成元 にこのアプリを開くURL（例: http://localhost:8080）を追加
    ※ file:// で直接開くとGoogle認証が許可されません。下記の方法でローカルサーバ経由で開いてください。
  6. 発行されたクライアントIDを上の欄に入力して「設定を保存」→「Googleドライブに接続」
ローカルサーバで開く例： このフォルダで次を実行し http://localhost:8080 を開く
python -m http.server 8080 （Pythonの場合）

認証が使えない環境でも、データはブラウザ内(localStorage)に保存され、上の「手動バックアップ」でドライブフォルダへの保存／復元が可能です。
