# komon-manager

Mykomonのような税理士向けの顧問先管理ツールです。`index.html` をブラウザ（Chrome推奨）で開くだけで動作します。

## 主な機能

- **顧問先情報**: 登録・確認・編集・削除、Excel出力、Excelインポート、インポート用テンプレートのダウンロード
- **案件進捗管理**: 案件ごとの進捗管理表（ステータス・進捗率・期限・担当者）、Excel出力
- **資料回収リスト**: 顧問先ごとの資料回収チェックリスト（未依頼／依頼済／回収済）、Excel出力
- **議事録**: 顧問先別の議事録保管、Excel出力
- **ダッシュボード**: 件数サマリと期限の近い案件の一覧

## データの保存

- 入力内容は自動的にブラウザ内（localStorage）に保存され、アプリを閉じて再度開いても続きから利用できます。
- 設定画面で **GoogleドライブのフォルダURL** と **OAuthクライアントID** を登録すると、そのフォルダに `komon-manager-data.json` として内容を自動同期します（複数端末での共有・バックアップに利用できます）。
- 認証を使わない場合でも、設定画面の「手動バックアップ」からJSONファイルとして書き出し／取り込みができます。書き出したファイルをドライブフォルダに置いておけば、別端末で取り込んで続きから作業できます。

## Googleドライブ連携の準備

`file://`（ファイルを直接ダブルクリック）で開いた場合、Googleの認証は許可されません。Drive自動同期を使う場合はローカルサーバ経由で開いてください。

```
# このフォルダで実行（Pythonがある場合）
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクトを作成
2. 「Google Drive API」を有効化
3. OAuth同意画面を設定し、自分のアカウントをテストユーザーに追加
4. OAuthクライアントID（ウェブアプリケーション）を作成し、承認済みのJavaScript生成元に `http://localhost:8080` を追加
5. アプリの設定画面にクライアントIDとドライブフォルダURLを入力 →「設定を保存」→「Googleドライブに接続」

## 技術

- 単一のHTMLファイル（依存ライブラリなしで動作）
- Excel入出力: [SheetJS](https://sheetjs.com/)（CDN読み込み）
- Googleドライブ連携: Google Identity Services + Drive API v3
