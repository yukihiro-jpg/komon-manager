# komon-manager

Mykomonのような税理士向けの顧問先管理ツールです。`index.html` をブラウザ（Chrome推奨）で開くだけで動作します。

## 主な機能

- **顧問先情報**: 登録・確認・編集・削除、納付方法（ダイレクト納付／紙納付書）、Excel出力／インポート／テンプレート
- **確定申告管理**: 決算月ごとに契約中の顧問先を自動集計（休止・解約は除外）。年度切替に対応し、決算月のカードを開くと申告書作成の進捗管理表（行＝対象法人、列＝工程、セルに完了日 or `未(期限)`）を表示。各法人の確定税額を入力すると翌期の中間申告を自動計算。
- **中間申告管理**: 暦月ごとの管理表（「〇年〇月中間申告」が12枚／年度切替）。確定申告管理で入力した中間情報から、その月に納付期限が到来する顧問先を決算月に関係なく自動集計し、法人税・地方法人税・県民税・事業税・市税・消費税（〇/〇回目）・合計額・納付方法・完了チェックを表示。
- **所得税／年末調整／法定調書／償却資産／その他 管理**: それぞれ年度切替付きの進捗管理表。工程（列）は編集可能、Excel出力対応。
- **毎月源泉税管理**: 毎月納付の顧問先を対象に、確定申告管理と同様に12枚の「○月分（翌月10日納付分）」カードを表示（年度切替対応）。カードを開くと月別の進捗管理表（顧問先名・所轄税務署・納付方法・給与ソフト・未済繰越額・システム登録・抽出＆送信・連絡方法・お知らせ・備考）が開きます。所轄税務署・納付方法・給与ソフト・連絡方法は顧問先情報から表示。
- **分析**: 顧問先情報から決算月別の件数、担当者別の件数・顧問料、区分別、契約ステータス別、納付方法別、顧問料合計／平均などを集計・グラフ表示。
- **AI（議事録の校正・まとめ）**: 議事録本文の校正、および顧問先ごと・事業年度ごとの議事録まとめを生成。決算時に当期の論点・決定事項を踏まえているか確認できます（顧問先一覧の「📋 まとめ」から）。AIプロバイダーは **Gemini（Google・無料枠あり）** と **Claude（Anthropic）** から選択でき、設定画面でAPIキー・モデルを登録します（既定はGemini）。Geminiのキーは [Google AI Studio](https://aistudio.google.com/apikey) で無料取得できます。
- **資料回収リスト**: 顧問先ごとのチェックリスト（未依頼／依頼済／回収済）、Excel出力
- **議事録**: 顧問先別の議事録保管（一覧では議事内容を表示）、Excel出力
- **ダッシュボード**: 件数サマリと期限の近い工程の一覧

UIはGoogleアプリ風（Material系）で、ナビゲーションは画面上部の横並びタブです。

### 中間申告計算のロジック（現行法・予定申告方式／前期実績基準）

税務申告ソフトの「翌期予定納付額」と一致するよう、端数処理を現行法に合わせています。

- **法人税割系（法人税・地方法人税・住民税法人税割・事業税・特別法人事業税）**: 前期確定税額 ÷ 前期月数（1円未満切捨）× 6（100円未満切捨）。
- **均等割（住民税）**: 年額 × 6 ÷ 前期月数（100円未満切捨）。
- **消費税（前期国税年額C・地方消費税を除く）**: 48万円以下＝なし／48万円超400万円以下＝年1回／400万円超4,800万円以下＝年3回／4,800万円超＝年11回。1回あたり国税＝C×(中間課税期間月数/前課税期間月数)、地方消費税＝中間国税×22/78。
- 法人税の予定申告額が10万円以下なら中間申告不要。
- 税額入力は **県民税（法人税割・均等割）／事業税／特別法人事業税／市民税（法人税割・均等割）** を区分して入力します（区分ごとに端数処理するため、ソフトの計算と一致します）。
- **申告納付期限**: 各中間課税期間の末日の翌日から2か月以内。決算月から逆算した暦年月で中間申告管理に反映。
- 確定申告管理の「確定法人税等」は法人税・地方法人税・県民税・事業税・特別法人事業税・市民税の年税額合計、「確定消費税」は消費税の国税＋地方消費税(国税×22/78)です。

### 進捗管理表の操作

- すべての進捗管理表（確定申告／所得税／源泉税／年末調整／法定調書／償却資産／その他／中間申告）で、**ヘッダをクリックで並び替え（昇順／降順）**、各列の **🔍欄で絞り込み検索** ができます。
- **中間申告管理の表示項目（列）は設定画面から任意に選択**できます。

## データの保存

- 入力内容は自動的にブラウザ内（localStorage）に保存され、アプリを閉じて再度開いても続きから利用できます。
- 設定画面で **GoogleドライブのフォルダURL** と **OAuthクライアントID** を登録すると、そのフォルダに `komon-manager-data.json` として内容を自動同期します（複数端末での共有・バックアップに利用できます）。
- 認証を使わない場合でも、設定画面の「手動バックアップ」からJSONファイルとして書き出し／取り込みができます。書き出したファイルをドライブフォルダに置いておけば、別端末で取り込んで続きから作業できます。

## Googleドライブ連携の準備

`file://`（ファイルを直接ダブルクリック）で開いた場合、Googleの認証は許可されません。Drive自動同期を使う場合はローカルサーバ経由で開いてください。

```
# このフォルダで実行（Pythonがある場合）
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクトを作成
2. 「Google Drive API」を有効化
3. OAuth同意画面を設定し、自分のアカウントをテストユーザーに追加
4. OAuthクライアントID（ウェブアプリケーション）を作成し、承認済みのJavaScript生成元に `http://localhost:8080` を追加
5. アプリの設定画面にクライアントIDとドライブフォルダURLを入力 →「設定を保存」→「Googleドライブに接続」

## 技術

- 単一のHTMLファイル（依存ライブラリなしで動作）
- Excel入出力: [SheetJS](https://sheetjs.com/)（CDN読み込み）
- Googleドライブ連携: Google Identity Services + Drive API v3
